# Gestor de Gastos - con framework Django


Aplicación web desarrollada con Django que permite:

- Registro y login de usuarios
  
- Gestión de gastos personales
  
- Mensajería interna
  
- Perfil de usuario

3. Instalar dependencias
   
4. Ejecutar migraciones
